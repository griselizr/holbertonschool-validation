
## How to Buil a Hugo Website
--------------
Prerequisites
--------------
-Install Hugo

-Install git

-Install Gnu makefile

## Steps
1.hugo new (insert name of site)

Construct the website

2.cd ( insert name of the site)

Change to root directory

3.echo "theme = 'ananke'" >> config.toml

Apply theme style

4.hugo server

Display website
